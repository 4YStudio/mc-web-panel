# 🚀 MC Web Panel - 为您的 Minecraft 服务器打造的现代化管理利器

还在为繁琐的 Minecraft 服务器管理感到头疼吗？是时候通过 **MC Web Panel** 升级您的管理体验了！

MC Web Panel 是一款专为 Fabric 服务器设计的**现代化、轻量级且高性能**的 Web 管理面板。基于 Node.js 和 Vue 3 构建，我们致力于为您提供极致顺滑的操作体验和赏心悦目的视觉设计。

## ✨ 为什么选择 MC Web Panel？

### 🎨 颜值与实力并存
告别那些陈旧、复杂的控制台。我们精心打磨的 UI 设计，不仅美观，更直观易用。
- **实时仪表盘**：一眼掌握服务器各项指标（CPU、内存、在线人数），实时监控控制台输出。
- **响应式设计**：无论是在电脑还是手机上，管理体验始终如一。

### 🛠️ 强大的即开即用功能
无需繁琐配置，核心功能一应俱全：
- **📂 文件管理**：网页端直接上传、下载、编辑配置、解压数据包。
- **👥 玩家管理**：轻松管理白名单、OP、封禁列表，一键踢出或传送捣乱玩家。
- **🧩 模组管理**：拖拽上传 Mod，版本管理从未如此简单。
- **🛡️ 安全无忧**：内置 Google 身份验证器（2FA）支持，为您的服务器安全保驾护航。

### 🔌 独家深度整合
我们不只是简单的管理面板，更为热门 Mod 提供了**原生级的图形化配置界面**，这是其他面板所不具备的绝杀功能！
- **🎙️ Simple Voice Chat (简单语音聊天)**：直接在面板里修改语音端口、密码，无需翻找配置文件。
- **🔒 Easy Auth (简单认证)**：图形化管理玩家注册信息，重置密码、注销用户，一切尽在掌握。
- **💾 Advanced Backup (高级备份)**：支持快照和增量备份，数据安全滴水不漏。

### ⚡ 极速部署，快人一步
没有复杂的依赖，没有繁琐的安装脚本。
1. **下载**: 获取对应系统的单文件执行程序（支持 Linux & Windows）。
2. **运行**: 双击或命令行启动。
3. **享受**: 打开浏览器，即刻开始管理！
